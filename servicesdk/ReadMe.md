第三方开发者使用的sdk
==================

Local Development
-----------------

Working on this project? Here's some helpful Gradle tasks:

 * `makeJar` - make jar. 包含所有接口文件的jar
 * `buildJar` - 给外部开发者使用的jar,只包含 "com/ubtechinc/alpha/sdk/**/*.class,com/ubtechinc/alpha/serverlibutil/**/*.class"
